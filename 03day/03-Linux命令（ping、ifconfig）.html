Linux命令(ifconfig,ping)

1 查看或配置网卡信息：ifconfig

ifconfig 是 linux 中用于显示或配置网络设备（网络接口卡）的命令

1 查看ip地址：

  windows 命令： ipconfig
   linux 命令： ifconfig

2 开启和关闭网卡

 ifconfig <网卡名称>  down|up
 比如关闭网卡： sudo ifconfig  ens33 down

3 修改网卡ip地址

  sudo ifconfig <网卡名称>  <新ip地址>
  比如：sudo ifconfig ens33   192.168.1.111

2 测试远程主机连通性：ping

ping 是 Windows、Unix 和 Linux 系统下的一个命令. 利用 "ping" 命令可以检查网络是否连通，可以很好地帮助我们分析和判定网络故障。

ping ip地址

 ping 192.168.1.1

ping 域名

ping www.baidu.com
ping stu.boxuegu.com
